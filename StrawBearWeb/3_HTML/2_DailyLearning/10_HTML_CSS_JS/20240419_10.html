12.使用现代CSS技术创建布局

12.1 准备进行页面布局

CSS盒子模型
外边距,边框,内边距,内容,是所有布局的基础,是每个元素的结构

渐进增强
一种设计思想,先从所需最少得网页开始,然后可以使用设计和交互元素来增强网页,首先注意力要集中在每个网页需要做什么?
所有的网页首先都需要可见性,功能性,可访问性.然后就可以通过CSS让他更加好看,通过JavaScript脚本让他具有交互作用

将设计中的结构和交互性分离
目标应该是创建网页,HTML代码仅包含结构和内容,CSS代码仅包含设计,JavaScript代码仅包含交互,职责分明.

12.2 移动设备优先的重要性

因为现在大多数人都使用移动设备进行观察,这是最大的人群,所以要优先考虑在移动设备上的效果如何.

12.3 理解固定布局

页面主体被设计成固定的宽度,所以会受到不同分辨率的影响.

12.4 理解流式布局

不使用固定宽度,尽可能的使用百分比宽度,无论用户的屏幕分辨率是多少,都能有比较好的效果.
