8.使用颜色,图像和多媒体


8.1 一些颜色选择建议

使用在生活中最常见的令人舒服的色彩,避免使用会导致眼睛疲劳的过亮颜色
使用具有两包对比度的颜色,易于阅读.避免使用对比度太低的组合
不需要太多个颜色,通常3-5种颜色,主色调占70%,补色25%,跳色5%.
Paletton配色网站

8.2 了解网络颜色

通过style="background-color: 颜色值;"即可设置对应的标签部分的背景颜色,如果在body标签设置则会应用于整个网页背景.

8.3 使用十六进制表示颜色

通过style="background-color: #999999;"通过十六进制表示的#rrggbb来表示颜色

8.4 使用RGB或RGBa值来表示颜色

通过style="background-color: rgb(128,128,128);"通过rgb来表示颜色

8.5 在CSS中指定背景,文本和边框颜色

例如可以通过
a
{
border: 1px solid #000; 指定边界的宽度,样式和颜色
background-color: #999999; 来指定所有超链接的背景都是灰色的
}
