11.使用CSS对列表,文本和导航进行更多操作

11.6 设置单层垂直导航的样式

通过设置CSS样式 

                nav ul  设置nav里面的ul标识
                {
                    list-style: none;设置不要标识符
                    margin:12px 0px 0px 0px;设置外边距上部12px
                    padding: 0px;设置没有内边距
                }
                nav li  设置nav里面的li标识
                {
                    border-bottom:1px solid #ffffff;设置底部有边框1px宽度,标准线条样式白色
                }
                nav li a:link,nav li a:visited  设置连接样式和显示样式
                {
                    font-size:15px;  字体尺寸
                    font-weight: bold;  字体样式加粗
                    display:block;  显示模式:块,每个元素都换行
                    padding:3px 0px 3px 3px;  设置内边距上,左下都有3px
                    background-color: #628794;  设置背景颜色
                    color:#ffffff;  设置文字颜色
                }
调整了整个的导航样式
