8.使用颜色,图像和多媒体


8.16 将图像转换为链接

通过在a标签对设置链接到大图的链接,在内设置一个img显示一个小的缩略图,这样就可以点击小图弹出大图了.

8.17 使用背景图像

background-color 指定元素的背景色
background-image 指定背景图片
background-repeat 指定背景图片的重复方式
background-position 指定初始位置

在CSS当中设置
body
{
background: #ffffff url("背景图地址") no-repeat top right;
}
