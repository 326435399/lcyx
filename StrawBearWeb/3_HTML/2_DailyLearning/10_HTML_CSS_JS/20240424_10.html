13.1 了解背景和边框

测试背景图片
background系列
background-image:用作背景的图像
background-position:图像放在元素上的位置
background-repeat:定义图像的重复方式
background-origin:背景图像开始平铺的原点
background-attachment:背景如何相对于视区移动
background-color:背景颜色
Tips:background: url(../0_Resoureces/img/gameScreen/封面.png)这个url是直接从当前CSS路径出发的

border: border-width border-style border-color;边框属性
solid:连续实线(默认值).
none:不绘制边框线
hidden:可绘制边框,但是隐藏
dashed:绘制虚线
dotted:绘制点线
double:绘制双线
groove:添加一个斜角
ridge:添加一个斜面
inset:添加一个轻微斜角看起来凹陷
outset:添加一个轻微的斜角看起来稍微凸起
