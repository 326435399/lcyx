8.20 嵌入多媒体文件

1.可以通过video controls src="视频地址" type="video/mp4"来播放嵌入视频

另外一种方式是通过video controls,然后在viedo标签对内新增多个source标签,里面会按照顺序执行
video controls
source src="视频地址" type=video/x-ms-wmv
source src="视频地址" type=video/mp4
写一段话,如果上面的都不行
/video
这样会提供更好的兼容性

video还有几个属性preload是否预加载,loop是否循环播放视频,poster视频还没有加载好现实的图像,autoplay准备好后立即播放
