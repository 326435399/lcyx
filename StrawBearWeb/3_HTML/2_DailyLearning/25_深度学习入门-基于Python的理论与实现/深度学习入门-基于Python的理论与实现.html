3.7 小结

介绍了神经网络的前向传播,感知机使用阶跃函数作为激活函数,神经网络使用sigmoid或ReLU函数.
使用Numpy多维数组可以提高运算效率
机器学习问题大体分为回归问题和分类问题,回归问题的激活函数使用恒等函数,分类问题使用softmax函数.
分类问题中输出神经元的数量等于分类数.
输入数据的集合称为合批.
